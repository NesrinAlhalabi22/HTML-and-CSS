# HTML-and-CSS

This repository contains two core web development projects that demonstrate proficiency in modern front-end technologies.

## 1. Bootstrap Cinema Project
A responsive movie theater website designed for Nesrin Alhalabi.
- **Integrated Bootstrap 5:** Utilizes the grid system for a responsive layout.
- **Card Decks:** Features custom movie cards for *Kong: Skull Island*, *Ghost in the Shell*, and *Beauty and the Beast*.
- **Pill Badges:** Specific "Info" colored badges used to display 6 unique showtimes per movie.
- **Navigation:** Fully functional bookmarks for smooth single-page navigation.

## 2. Professional Portfolio (HTML & CSS Only)
A unique, dark-themed portfolio highlighting custom CSS skills.
- **Video Hero:** Uses an autoplaying, muted, looped video of running code with a high-contrast text overlay.
- **Image Manipulation:** Displays a centered creative image with custom CSS filters (grayscale/contrast) and rounded corners.
- **Interaction:** Includes a sophisticated image overlay fade effect and hover transitions on text and navigation links.
- **Layouts:** A robust two-column layout using modern Flexbox and a fixed top menu.
- **Data Display:** A fully styled table with cell padding and a highlighted header row for course tracking.

## Technologies Used
- **HTML5:** Semantic structure, video embedding, and table data.
- **CSS3:** Flexbox, CSS Filters, Positioning, and Keyframe Animations.
- **Bootstrap 5:** Component styling and responsive grid.
